## Pair with given sum in a sorted array

### Solution :

- Time Complexity :


```java
int Countpair(int a[], int n, int sum)
{
    int count = 0;
    HashMap<Integer, Integer> map = new HashMap<Integer, Integer>();

    int i;
    for(i=0; i<n; i++)
    {
        if(map.containsKey(sum - a[i]))
        {
            count++;
        }
        map.put(a[i],i);
    }
    if (count == 0)
        return -1;

    return count;
}

```
