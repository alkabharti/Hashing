## Largest subarray with 0 sum

## Solution :

- Time Complexity :


```java
int maxLen(int arr[], int n)
{
    HashMap<Integer, Integer> map=new HashMap<>();
    int max_len=0, sum=0;
    int i;
    map.put(sum,-1);
    for(i=0;i<n;i++)
    {
        sum+=arr[i];
        if(!map.containsKey(sum))
        {
            map.put(sum,i);
        }
        else
        {
            max_len=Math.max(max_len,i-map.get(sum));
        }
    }
    return max_len;
}


```
